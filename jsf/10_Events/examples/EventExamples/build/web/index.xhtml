JSF Event Model Examples

Application Events

Goal: Display the syntax and different ways of handling application events, that are value change and actionevents

Value Change Events

Value change events are fired by editable value holders—such as h:inputText, h:selectOneRadio, and h:selectManyMenu —when the component’s value changes

Output Source Code Description
/index_1 index_1.xhtml A JSF page that captures a value change event in, h:inputText inputzipby using the valueChangeListener atrribute. Note the use of binding, immediate and onchange attributes.
AutoFillBean.java The backing bean for the JSF Address form
/index_2 index_2.xhtml A JSF page that captures a value change event in, h:selectOneMenupby using the valueChangeListener atrribute.
CountryBean.java The managed bean for the JSF form
/index_3 index_3.xhtml A JSF page that captures a value change event in, h:selectOneMenupby using the f:valueChangeListener tag
CountryBean.java The managed bean for the JSF form
CountryValueListener.java The user defined Listener class for the value change event in the h:selectOneMenu component

Action Events

Action events are fired by action sources—for example, h:commandButton and h:commandLink—when the button or link is activated.

Output Source Code Description
/index_4 index_4.xhtml A JSF page with a command button with a actionlistener attribute whose value performs the event handling for a click action on the button
NormalBean.java The managed bean for the JSF form
/index_5 index_5.xhtml A JSF page with a command button with a f:actionlistener tag. The class name in the type attribute performs the event handling for a click action on the button
NormalBean.java The managed bean for the JSF form
NormalActionListener.java The user defined Listener class for the action event in the h:commandButton component

Lifecycle Events

Goal: Display the syntax and different ways of handling lifecycle events, that are phase and systemevents

Phase Events

The JSF implementation fires events before and after each life cycle phase

Output Source Code Description
/index_6.xhtml index_6.xhtml A JSF file with a form to select a specific phase that also has a valueChangeListener set for the selectOneListBox
FormBean.java The managed bean for the form
PhaseTracker.java The PhaseListener class that implements the PhaseListener interface
faces-config.xml The configuration file where you register the PhaseTracker class which is the PhaseListener in this example
