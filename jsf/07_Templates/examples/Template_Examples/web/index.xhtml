Templating Tags

Goal: Display the syntax and purpose of using templating tags like ui:compostion, ui:define, ui:insert, ui:include, ui:decorate.

Output Source Code Description
/page1 page1.xhtml A JSF page that uses a template(commonLayout.xhtml) by using , ui:composition and ui:define tags . It also overrides the common content and common footer files to provide its own content and footer. It uses the defualt header provided by the template.
commonLayout.xhtml The master template file that used ui:insert and ui:include tags
commonHeader.xhtml This file is included in the master template file as a header
commonContent.xhtml This file is included in the master template as content
commonFooter.xhtml This file is included in the master template as a footer

ui:decorate and ui:param tags

Goal: ui:decorate does not ignore all the content outside of the tag and ui:param is used to define name value pairs.

Output Source Code Description
/decorator_eg decorator_eg.xhtml A JSF page shows that the content outside ui:decorator tag is considered and ui:include uses the ui:param tage
masterDecorator.xhtml The template file
User.java ui:param tag has details as the name and user as its value that passes to the file in the ui:include tag, welcome.xhtml
welcome.xhtml This file is included in the decortaor_eg.xhtml by using the ui:include tag outside the ui:decorate tag
